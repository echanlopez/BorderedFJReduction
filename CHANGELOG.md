## [1.0.0] – Initial public release

### Added
- Symbolic implementation of the Faddeev–Jackiw reduction using geometrically constrained matrix bordering.
- Iterative constraint propagation with explicit rank diagnostics.
- Automatic detection of gauge symmetries via null-space analysis.
- Structured symbolic object representing the reduced system.
- Visualization tools for extended symplectic structures.

### Notes
- This release establishes the core theoretical and software architecture of the package.
- Intended for analytical and research use on finite-dimensional constrained systems.
